## Getting Started Todo-App v2

cp .env.example .env <br />
composer install <br />
php artisan key:generate <br />
npm install && npm run dev <br />
create a database todo-app-v2 <br />
php artisan migrate --seed <br />
run the project php artisan serve <br />
